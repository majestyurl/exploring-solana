The Basics...

Before we can get into exploring the Solana blockchain,
we first have to get an understanding of what exactly.

A blockchain is essentially a ledger - a series of
records and data points, like a spreadsheet or accounting book.
The key difference between a blockchain & traditional spreadhseets
or accounting systems, is that blockchains are decentralized,
meaning instead of the information being stored in one place,
the information is stored accross multiple computers around the world
connected to a network.

This is beneficial because there is no single point of failure, meaning
if a computer on the network is compromised, the data can be verified
by referencing the data stored on the other computers on networks.

There are different blockchains people can use for different reasons.
Blockchains can be used to store data, value, or in Solana's case,
run programs.

Cryptography

Cryptography is the art of keeping things secure - think
"encryption" or passwords. Cryptocurrency is a type of
currency based on cryptography. Transactions are decentralized
and as secure as any other encrypted data. What does this mean?
Its a very safe way of storing data and value with no choke points
or single points of failure and is very hard to manipulate.

Coins vs Tokens

What is the difference between a token and a coin? All blockchain
protocols have coins that represent value and can be traded. On Solana
this coin is called "SOL".

Tokens, on the otherhand, are built on top of a base blockchain protocol
(layer one). All projects built on a blockchain are tokenized and the tokens
are used to do things on the blockchain through trading, including
executing smart contracts.

Smart Contracts

What exactly is a "smart contract"?

A smart contract is a contract written into the code set to
automatically execute when certain conditions are met. Blockchains
like Solana or Ethereum are capable of runnig smart contracts,
but Bitcoin cannot.

Dapps

Dapps are decentralized apps, apps that have smart contracts
built into them.
